# pet_projects

---

В данном репозитории представлены материалы и отчетная документация по двум курсам. 

### Анализ и проектирование на UML
Полный цикл проектирования информационной системы. Реализованы все 7 лабораторных работ, включающие:
* **Функциональное моделирование:** Диаграммы использования (Use Case) и деятельности (Activity).
* **Статическое описание системы:** Концептуальные и технические диаграммы классов (Class Diagram), спроектированные с учетом многослойной архитектуры.
* **Динамическое моделирование:** Диаграммы последовательности (Sequence) и состояний (State Machine) для описания логики взаимодействия объектов.
* **Физическая архитектура:** Модели развертывания (Deployment Diagram) с описанием узлов и артефактов системы.

### Управление требованиями (УТ)
Документирование и формализация спецификаций программного продукта. 
* **Выполненные этапы:** Сбор и классификация требований, верификация по критериям качества (атомарность, выполнимость, тестируемость), приоритизация методами MOSCOW/RICE и формирование матрицы трассировки (RTM).
* **Текущий статус:** Разработка завершающих этапов — регламентация процесса управления изменениями (ЛР 4) и подготовка итоговой спецификации SRS по стандарту IEEE 830 (ЛР 5).
